# プライバシーポリシー

**毒物劇物Pass**（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めます。本プライバシーポリシーは、本アプリにおける情報の収集・利用・管理方針について説明するものです。

---

## 1. 収集する情報

本アプリは以下の情報を収集・利用します。

### アプリ内で保存する情報

- **学習データ**：問題への回答履歴、正誤記録、苦手フラグ（ブックマーク）、連続学習日数、リワード広告による問題解放履歴
- **設定情報**：文字サイズ設定、試験区分（一般・農業用品目）設定、都道府県設定

これらの情報は、お使いの端末内（ローカルデータベースおよび端末内ストレージ）に保存されます。

### Firebase Analytics（利用状況分析）

本アプリは、利用状況の把握と品質改善のために Firebase Analytics を利用します。Firebase Analytics では、以下のような情報が収集される場合があります。

- アプリの起動、学習開始、回答、学習完了などの利用イベント
- アプリのバージョン、ビルド番号、OS 種別などのアプリ・端末環境情報
- 画面や機能の利用状況
- アプリの動作状況やパフォーマンスに関する診断情報（Firebase Analytics が提供する範囲）
- おおよその地域情報、デバイス識別子等、Firebase Analytics の仕様上収集される可能性がある情報

本アプリでは、Firebase Analytics を利用状況分析と品質改善目的のみに使用します。広告 ID を第三者横断トラッキング目的で利用せず、iOS では IDFA を利用しない構成としています。

Firebase のプライバシーとセキュリティに関する情報は、Google のポリシーをご確認ください。

Google プライバシーポリシー：[https://policies.google.com/privacy](https://policies.google.com/privacy)

### Firebase に送信しない情報

本アプリは、Firebase Analytics に以下の情報を送信しません。

- 問題本文
- 選択肢本文
- 解説本文
- 自由入力
- 記述答案
- 氏名
- メールアドレス
- 個別相談内容
- 課金管理サービスの SDK キー
- 広告ユニット ID
- Apple の取引 ID 等の決済識別子の生値
- 問題 ID
- 課金管理サービスのユーザー ID
- 公式問題本文や出題文を復元できる情報

### 今回利用しない Firebase サービス

本アプリでは、Firebase Analytics 以外の Firebase サービスは利用しません。具体的には、Firebase Authentication、Cloud Firestore、Realtime Database、Cloud Functions、Storage、BigQuery export、Firebase AI Logic、Remote Config、Crashlytics、App Check、Cloud Messaging は利用しません。

### Google AdMob（広告配信）

無料ユーザー向けに広告を表示するため、Google AdMob を利用しています。AdMob は広告配信のために以下の情報を収集する場合があります。

- デバイス識別情報（広告 ID 等）
- IP アドレス
- アプリの使用状況
- 広告の表示・操作に関する情報

AdMob のプライバシーポリシー：[https://policies.google.com/privacy](https://policies.google.com/privacy)

### RevenueCat（購入管理）

プレミアムプランの購入状態を管理するため、RevenueCat を利用しています。RevenueCat は、購入状態の確認、復元、管理のために必要な情報を処理する場合があります。

RevenueCat のプライバシーポリシー：[https://www.revenuecat.com/privacy](https://www.revenuecat.com/privacy)

---

## 2. 情報の利用目的

収集・保存した情報は以下の目的にのみ利用します。

- 学習進捗の管理・表示（正答率、今日の学習記録、苦手科目の分析）
- 誤答優先学習、苦手問題モード、本番想定モードなどの学習機能の提供
- 連続学習日数、リワード広告による問題解放履歴の管理
- アプリ設定の保持
- プレミアムプランの購入状態の確認・復元
- 広告の配信（無料ユーザーのみ）
- アプリの利用状況分析および品質改善

---

## 3. 情報の第三者提供

本アプリは、法令に基づく場合を除き、ユーザーの情報を第三者に販売することはありません。

本アプリは、上記の目的の範囲内で、Firebase Analytics、Google AdMob、RevenueCat などの第三者サービスを利用します。各サービスにおける情報の取り扱いは、それぞれのプライバシーポリシーに従います。

---

## 4. トラッキングについて

本アプリは、iOS における App Tracking Transparency（ATT）の許可を要求しません。IDFA を使用せず、Firebase Analytics を第三者のアプリや Web サイトをまたいでユーザーを追跡する目的では利用しません。

---

## 5. 情報の保管と削除

学習データはお使いの端末内に保存されます。アプリ内の「設定」→「学習データをリセット」から、回答履歴および苦手フラグ等の学習データを削除できます。アプリをアンインストールすることで、端末内に保存されたローカルデータは削除されます。

第三者サービスにより処理される情報の保管期間や削除方法については、各サービスのプライバシーポリシーをご確認ください。

---

## 6. 未成年者の利用について

本アプリは特定年齢層を対象とした利用制限は設けていませんが、未成年者が利用する場合は保護者の同意のもとでご利用ください。

---

## 7. プライバシーポリシーの変更

本プライバシーポリシーは、法令の改正やサービス内容の変更に応じて予告なく改定する場合があります。改定後のポリシーは本ページに掲載した時点から効力を生じるものとします。
